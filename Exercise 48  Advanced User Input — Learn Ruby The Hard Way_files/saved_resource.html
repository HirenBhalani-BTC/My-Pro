                      zedshaw Mod BasarErdivanli • 2 years ago

                      Ultimately, you should go with whatever is simpler for the task. The difference between to_i and Integer() is simply that you can tell if someone did not enter a number with Integer(). With to_i you get a 0 if they enter anything, so "fasdfasdfadsf".to_i will be 0 as well as "0".to_i.

                      If you use Integer though, just return it. There's no need to do to_i after that.
